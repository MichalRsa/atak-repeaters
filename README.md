Lista przemienników pobrana ze strony [przemienniki.net](https://przemienniki.net/) w formacie .kmz. Przystosowana do wyśweitlania listy w aplikacji ATAK.

1. [Pobierz plik repeaters.kmz](repeaters.kmz)
   ![Pobieranie pliku](https://github.com/user-attachments/assets/9401a533-c752-4a0f-a45b-ee374ef4f96b)

3. W aplikacji ATAK
    - importu plik przy pomocy narzędzia 'Import'.
    - plik dostępny będzie w zakładce 'Overlay Manager > File Overlays > Mapa przemienników | przemienniki.net'
